Technology

Chinchilla was made using node.js and the iTunes API. All tracks are being played via YouTube Player.
These open source libraries were used in this project. Thanks!

  • Express
  • itunes-search
  • jsonreq
  • mongoskin
  • socket.io
  • swig
  • underscore.js and underscore.string
